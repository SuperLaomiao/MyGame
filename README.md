# 俄罗斯方块小游戏 🎮

一个经典的俄罗斯方块游戏，使用 HTML、CSS 和 JavaScript 开发。

## 游戏特性

- ✅ **经典玩法** - 完全遵循俄罗斯方块传统规则
- ✅ **7种方块** - 包含所有标准俄罗斯方块形状
- ✅ **键盘控制** - 支持方向键和空格键
- ✅ **实时统计** - 显示得分、等级、消除行数
- ✅ **下一个方块预览** - 可以看到下一个方块
- ✅ **美观界面** - 现代设计风格

## 控制方式

- **← →** - 左右移动
- **↑** - 旋转方块
- **↓** - 加速下落（+1分/步）
- **空格键** - 直接落底（+2分/格）
- **P键** - 暂停/继续

## 如何运行

2. 选择 **"Download ZIP"**

1. 访问 `index.html` 文件
2. 点击右上角的 **"Raw"** 按钮
